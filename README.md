# 💰 投资界（Pedaily）Writer Skill

> 让 AI 以投资界「专业资本视角」风格写创投深度报道的可复用 Skill。

## 简介

本 Skill 基于对投资界（Pedaily）数百篇创投报道的结构化解析，将「资本视角+数据翔实+行业全景」的专业写法标准化为 AI 写作指令，面向 PE/VC 机构、创业者和 IPO 从业者。

## 文件结构

```
pedaily-writer/
├── SKILL.md                          # 主技能文件
└── resources/
    ├── playbook_funding.md           # 骨架一：融资事件深度报道
    ├── playbook_fund.md              # 骨架二：基金动态分析
    ├── playbook_ipo.md               # 骨架三：IPO与退出分析
    ├── playbook_market.md            # 骨架四：一级市场季报
    └── playbook_gp_lp.md             # 骨架五：GP/LP关系与募资
```

## 投资界核心写作风格

| 维度 | 特征 |
|------|------|
| 视角 | 资本视角（LP/GP/退出路径） |
| 信息 | 融资事件数据框前置（金额/轮次/领投方/估值） |
| 深度 | 分析投资逻辑，不只报道事实 |
| 语言 | 正式严谨，适合机构读者 |
| 禁忌 | 信源单一（来自公司PR）、忽视融资风险 |

## 核心写作哲学

- **资本视角优先**：从 LP/GP 和退出路径角度理解每一笔融资
- **数据先行**：融资事件必须有规范的数据框（金额/估值/机构）
- **行业全景**：不孤立报道单一事件，放入行业格局分析
- **风险平衡**：融资不只是好事，必须评估烧钱速度和退出压力

## 如何使用

```
使用 pedaily-writer，帮我用投资界风格写一篇关于 [融资事件/创投市场] 的分析报道
```
