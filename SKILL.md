# 投资界（Pedaily）风格写作 Skill

## 账号画像

**投资界 Pedaily** — 中国股权投资与创投信息服务平台
- 核心定位：一级市场最全面的投融资信息与分析平台
- 受众：PE/VC机构、LP、创业者、产业基金、IPO从业者
- 调性：**专业严谨 + 数据翔实 + 资本视角 + 全景覆盖**
- 口号精神：「中国创业投资第一门户」

---

## 核心写作特征

### 1. 标题公式

```
【融资快报型】
→ "[公司]完成[金额][轮次]融资，[领投方]领投"
→ "[N]家[赛道]公司密集融资：[共同主题]"

【资本分析型】
→ "[基金名]年内已投[N]个项目：[投资逻辑]"
→ "[季度/年度]创投数据：[核心趋势判断]"
→ "深度：[赛道]的钱还够烧多久"

【IPO/退出型】
→ "[公司]冲刺[交易所]IPO：[关键数据]一览"
→ "[年度]最赚钱的10笔创投退出：背后故事"
```

### 2. 文章结构

```
【数据摘要框（置于文章顶部）】
投资界特色：用数据框快速交代核心信息：
「融资概况：
  公司：[名称] | 轮次：[A/B/C轮] | 金额：[X亿]
  领投：[机构名] | 跟投：[机构1、2、3]
  估值：[X亿美元/人民币]（如有）
  用途：[资金用途]」

【公司现状：让数字说话】
- 核心业务指标（GMV/ARR/DAU/市占率）
- 与竞品的横向对比
- 融资历史（本轮之前的融资情况）

【投资人视角：资本为什么押注】
- 领投机构的历史投资组合分析（这次和他们过去的押注一致吗）
- 行业赛道吸引力（市场规模/增速/政策）
- 本次投资的「时间点逻辑」（为什么是现在）

【创业者视角：资金用途的战略含义】
- 钱打算花在哪里？（研发/扩张/团队/合规）
- 这轮融资要解决的核心战略问题是什么？
- 距离下一轮（或IPO），还需要证明什么？

【市场格局影响】
- 这笔融资对同赛道竞争对手意味着什么？
- 是否会触发竞争响应（跟投/反融资/并购）？
```

### 3. 写作风格原则

**必须做到**：
- 所有融资数据来源必须可核实（官方公告/权威媒体）
- 必须包含对投资机构背景的专业分析
- 资本视角优先（从LP/GP/退出路径角度思考）
- 语言正式，适合机构读者（不口语化）

**绝对不能**：
- 把融资作为纯利好宣传（要分析实质意义）
- 只写一家公司，不做行业横向对比
- 信源单一（主要来自公司PR的文章）
- 忽视融资风险和烧钱速度

## STEP 流程

### STEP 1：确认参数
```
1. 报道类型：单次融资事件 / 季度数据报告 / 基金动态 / IPO分析？
2. 核心数据是否已有（融资金额/轮次/领投方）？
3. 文章视角偏向：投资机构视角 or 创业者视角？
4. 字数：800（快报）/ 2000（深度）/ 3000（研报）
```

### STEP 2：构建「资本分析框架」
- 行业赛道评分（市场规模/竞争格局/政策环境）
- 公司竞争优势分析（技术/渠道/团队/数据）
- 退出路径评估（IPO/并购/战略投资）

### STEP 3：数据采集
重点渠道：
- IT桔子、企查查、天眼查（融资历史）
- 国家统计局、行业协会（市场数据）
- 交易所官网（IPO/上市公告）
- 基金官网/公告（机构历史投资组合）

### STEP 4：按骨架写正文

### STEP 5：自检清单
```
□ 文章顶部是否有融资概况数据框？
□ 是否分析了领投机构的历史投资逻辑？
□ 是否包含赛道横向竞争格局分析？
□ 是否讨论了资金用途的战略含义？
□ 所有数据是否有可核实来源？
□ 是否评估了融资后的关键风险或挑战？
```
