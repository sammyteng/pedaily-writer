# 骨架一：融资事件深度报道

## 数据框模板（必填）
```
融资概况：
公司：[名称] | 成立时间：[年份] | 注册地：
轮次：[Pre-A/A/B/C轮] | 金额：[X亿元/美元]
领投：[机构名] | 跟投：[机构1、2、3]
估值：[X亿美元]（如有披露）
用途：[资金用途概述]
```

## 结构模板
```
【融资概况框】→ 投资机构背景分析 → 公司核心数据 → 市场格局影响 → 退出路径评估
```

# 骨架二：基金动态分析

适用场景：某知名VC/PE基金的新基金募集、策略调整或投资方向变化。

## 结构模板
```
【基金概览框】
基金名称 | 目标规模 | 当前关闭规模 | LP来源 | 投资策略

【为什么现在募这笔钱】
时机分析：市场环境 + 基金自身节奏

【投资组合分析】
过去3年投了哪些项目？赚钱了还是亏损了？

【这期基金和上期有什么不同】
策略变化（赛道/阶段/地域/规模）及背后逻辑

【LP的期望与现实】
投资者关系角度：这个基金给LP的回报预期是什么
```

# 骨架三：IPO与退出分析

## 结构模板
```
【IPO基本信息框】
拟上市交易所 | 发行规模 | 估值 | 主承销商 | 进入审核阶段

【公司财务健康度】
核心财务指标（营收/净利润/现金流），与同类上市公司对比

【投资人退出分析】
谁持大股？持仓成本大约多少？IPO价格意味着多少倍回报？

【IPO风险评估】
上市面临的合规/市场/监管风险

【市场估值参照系】
同类上市公司PE/PS倍数对比，这个IPO估值是否合理
```

# 骨架四：一级市场季报

## 结构模板
```
【本季度关键数字】
融资事件总量 | 总金额 | 单笔最大融资 | 最活跃赛道 | 最活跃机构

【热门赛道分析】
Top3赛道的投资逻辑梳理

【冷清赛道与原因】
哪些领域资金在撤退？为什么？

【机构行为分析】
哪些基金在加速出手？哪些在收缩？

【下季度展望】
基于当前信息的可能方向（审慎表达）
```

# 骨架五：GP/LP关系与募资

## 结构模板
```
【LP市场当前状态】
国内LP结构（市场化母基金/政府/家办/险资）及活跃度变化

【GP与LP的博弈】
条款、回报分成、DPI（分配率）谈判动态

【募资困难or容易的背后原因】
结构性分析：为什么某类基金好募，某类难募

【GP如何提升对LP吸引力】
最佳实践分析
```
